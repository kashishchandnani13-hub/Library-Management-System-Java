## Library Management System
## Overview
This is a console-based Library Management System developed in Java. The project demonstrates the implementation of Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and collection handling using ArrayList.
The system allows users to manage books in a library through a menu-driven interface.

## Features
1. Add new books
2. Remove existing books
3. Search books by title
4. Display all books
5. Update book details
6. Borrow books
7. Check book availability
8. Prevent borrowing of already issued books
9. Case-insensitive book search

## Technologies Used
1. Java
2. Object-Oriented Programming (OOP)
3. ArrayList Collection Framework
4. Console-Based User Interface
5. Classes
6. Book Class

## Stores information about each book:
1. Title
2. Author
3. Borrow Status
